# 1日シミュレーション実験ログ

本番ロジック（screener→指標→ニュース→判断→約定→評価）を、過去日の市場データで
1日だけ動かし（初期100万円）、損益とロジックの妥当性・改善点を検証した記録。

実行: `scripts/simulate-day.ts`（build=口座リセット＋市場文脈生成 / execute=始値約定・終値評価）。
判断は GROQ ではなく人手（Claude）が本番ルールで実施。約定=当日始値・評価=当日終値の日足近似。

前提: 日中データなし・先読み回避（指標は前日まで）・ニュースは取得できた分のみ。

---

## 実験1: 2026-05-27（米国・USDJPY≈161.69）

### ロジック改善前（priceOverride/クランプのみ・スクリーナー無フィルタ）
- 採用: POWI 14 / NOK 70 / ON 9（いずれも上昇トレンド順張り）
- 結果: **-13,242円（-1.32%）**。3銘柄とも寄り→引けで下落。
- 診断:
  - スクリーナーが `|騰落率|×log(出来高)` のみで、候補がほぼ過熱(RSI≥70)・低位株($0.40等)・落ちるナイフ。
  - 「始値約定」が上ギャップの失速を掴みやすい（POWI 寄り+4.9%→失速）。
  - 「現金を遊ばせるな」の指示が、地合いの悪い日に無理な順張りを誘発。

### ロジック改善後（スクリーナーに株価$5・売買代金$5Mフロア／寄り+3%超ギャップ回避／プロンプト明記）
- 候補から低位株(MAXN/EDIT/SPCE等)が消え、健全銘柄が浮上。
- 採用: NVAX 110 / ON 9 / NOK 70 / DIOD 11。POWI(+4.9%ギャップ)は**自動見送り**、SOXL(+7.5%ギャップ・3倍ETF)も回避。
- 結果: **-1,482円（-0.15%）**。改善前比 **+11,760円**。
  - NVAX +10,849 / ON -5,893 / NOK -3,622 / DIOD -1,690。
  - ギャップガードが前回の負け銘柄POWIを排除、暴落したSOXL($242→$218)を回避できたのが効いた。
- コミット: `30dbb90`（ハーネス）, `10d6fc4`（フロス除外・ギャップ回避）

## 実験2: 2026-04-27（米国・USDJPY≈161.69）

実験1の改善（フロア・ギャップ回避）を適用済みの状態から開始。候補が極端に過熱
（RSI 77〜100が大半、SOXSはRSI0で暴落中）で、健全な買い場がほぼ無い地合い。

### 現行ルール（パラボリック・ガード無し）
- 「上昇トレンド＋RSI<70＋ギャップ≤3%」の唯一の適格が SKLZ だが、20日 **+214%** のパラボリック。
- 採用: SKLZ 120。結果: **-28,943円（-2.89%）**。SKLZが寄り$7.84→引け$6.36（-19%）と急落。
- 新たな弱点: 下方向の「落ちるナイフ回避」はあるが、**上方向の過熱(パラボリック)回避が無い**。
  RSI70未満でもモメンタムが過大な高騰し切り銘柄を掴んでしまう。

### 改善後（パラボリック・ガード追加: 20日モメンタム>+80%の新規BUYを自動見送り）
- SKLZ(+214%)はガードで**自動見送り**。他は全て過熱/下降トレンド/ギャップで不適格。
- 適格な順張りが1つも無いため**全額現金で保持**。
- 結果: **+0円（0.00%・資本保全）**。現行比 **+28,943円**。
- 学び: フロスだらけの日は「無理に順張りせず現金保持」が正解。パラボリック・ガードと
  「地合いが悪い日は現金保持可」の指示でそれが実現できた。
- 反映: `scripts/simulate-day.ts`（MOM_UP_MAX=80の自動ガード）, `lib/claude.ts`（プロンプト明記）

## 実験3: ランダム営業日バッチ（改善後ロジックのロバスト性）

`simulate-day.ts batch <N> <seed>` で米国の実営業日をランダム抽出し、改善後ロジック
（判断は `ruleDecide`＋gap/parabolic自動ガード）で1日収支を一括検証。各日は独立に初期
100万円から開始・メモリ内シミュレート（クランプは本番同等）・FXは現在レート固定近似・
ニュース不使用。

### 所見（合計16日）
- **資本保全は機能**: 適格な買い場が無い日は0取引で現金保持し、下落日にSPYを上回る（12-15, 12-18）。
- **明確なαは無い**: 平均ではほぼ市場追随〜やや劣後。1日収支は取引した2〜3銘柄の寄→引の
  方向に支配され、実質コインフリップ。集中が外れると-2〜-4.5%の日が出る（11-24, 03-06, 03-20, 06-05）。
- **次の改善方向**: 1日の方向性リスク（集中）の低減＝ポジションを小さく分散、または日中
  での利確/損切り。ただし日中データが無いため日足近似では検証不可。現状ロジックの価値は
  「フロスを買わない・地合い不良時は休む」というリスク回避面に最も表れている。

## 改善ルールの到達点（本番ロジックに反映済み）
1. ポジションサイズの自動クランプ（注文の全却下→0取引バグの解消）
2. スクリーナーのフロス除外（株価$5・売買代金$5Mフロア）
3. 落ちるナイフ回避（下方向: 売られすぎ＋モメンタム床割れは買わない）
4. パラボリック回避（上方向: 20日+80%超の高騰し切りは買わない）
5. 上ギャップ回避（寄り+3%超は失速を掴みやすい）
6. 低位株・損切り後クールダウン・地合い不良時の現金保持
